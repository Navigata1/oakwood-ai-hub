HDCP Issue — ZV Pro Broadcast Blocker

Problem

The ZeeVee ZV Pro QAM Broadcaster at 10.1.240.4 reports "No Video" because the HDMI source signal is HDCP-encrypted. The ZV Pro cannot process HDCP-protected content. Verified: same HDMI signal displays correctly on a standard monitor.

Proven Fix (Jimmy's church install)
Source—HDMI—>HDCP-Stripping Splitter—HDMI (clean)—>ZV Pro (.4)

Order 2–3 powered 1x2 HDMI splitters and test. Same approach working at church installation for 2+ years. Recommended: ViewHD VHD-1X2MN3D, OREI HD-102, EZCOO 1x2 4K, or HDFury Integral 2 (premium).
