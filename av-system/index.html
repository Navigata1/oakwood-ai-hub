1. 📷 PTZ Camera Control — Overview

✅ Bottom line: The V-80HD controls up to 8 PTZ cameras simultaneously over LAN via the DIRECT STREAM port. Full pan, tilt, zoom, focus, exposure, and preset recall — all from the board.

Supported Camera Brands & Protocols

BrandProtocol SettingFunctions
JVCJVCPTZ, Focus, Exposure, Preset, Tally
PanasonicPanasonicPTZ, Focus, Exposure, Preset, Tally
CanonCanon PTZPTZ, Focus, Exposure, Preset, Tally
SonyVISCA over IPPTZ, Focus, Exposure, Preset, Tally
PTZOpticsPTZOpticsPTZ, Focus, Exposure, Preset, Tally
AvonicAvonicPTZ, Focus, Exposure, Preset, Tally
KEY You can mix different brands at the same time. Camera 1 can be Sony, Camera 2 can be Panasonic, etc. The V-80HD handles all protocols simultaneously.

2. 🌐 Network Setup for PTZ Cameras

Manual p.99 & p.151

Prerequisites

  • Cameras and V-80HD must be on the same network/subnet
  • Each camera needs a static IP address configured on the camera itself
  • Connection goes through the DIRECT STREAM port (LAN/Ethernet)

V-80HD Network Configuration

[MENU] → Network → LAN Setup

TIP Use Network → Network Information to verify your connection status (Link Status, assigned IP, etc.)

3. 🔧 Registering & Configuring Cameras

Manual p.99 & p.152

Step-by-Step: Add a Camera

[MENU] → Camera Control

1
Select Camera ID → Choose Camera 1 through Camera 8
2
Set Protocol → Select the matching protocol for your camera brand:
N/A | JVC | Panasonic | Canon PTZ | VISCA over IP | PTZOptics | Avonic
3
Set IP Address → Enter the camera's static IP (e.g. 192.168.0.101)
4
JVC only: Enter Login Name and Password (JVC cameras require authentication)
5
Set Tally Channel → Specify which input connector (HDMI 1–4, SDI 1–4, SRT In) the camera's video comes in on. This enables the camera's tally light when it's live.
6
Press [MENU] to save and exit
IMPORTANT Repeat for each camera (up to 8). Each camera gets its own Camera ID, Protocol, IP, and Tally Channel assignment.

4. 💾 Saving & Recalling Camera Presets

Manual p.99–100

Each camera supports 8 presets. Presets store: pan position, tilt position, zoom level, focus, and exposure. Presets are saved in the camera itself (not in the V-80HD), so they persist even if the board is powered off.

Saving a Preset

1
[MENU] → Camera Control → Camera ID → Select the camera
2
Adjust Pan, Tilt, Zoom, Focus, Exposure to the desired position (hold [VALUE] knob to move)
3
Set Pan/Tilt Speed (1–24) to control how fast the camera moves
4
Select Camera Preset → Choose Preset 1–8
5
Select Store → Press [VALUE] knob → Confirm "OK"

Recalling a Preset (Single Camera)

1
[MENU] → Camera Control
2
Set Camera ID → Select the camera
3
Set All Cameras Recall → OFF
4
Select Camera Preset → Choose the preset number (1–8)
5
Select Recall → Press [VALUE] → "Completed."

Recalling Presets from ALL Cameras at Once

1
Set All Cameras Recall → ON
2
Select Camera Preset → Choose preset number
3
Select Recall → ALL cameras jump to their respective preset at that number

💡 This is HUGE for church production — set Preset 1 as "wide shot" on every camera, Preset 2 as "podium close-up", etc. One button = every camera moves.

Manual p.97 & p.141–142

This is how you eliminate the joystick. The V-80HD has 8 assignable pads × 8 banks (A–H) = 64 assignable functions. You can dedicate an entire bank to camera control.

How to Set Up Pads

1
Press ASSIGNABLE PADS [SETUP] button
2
Select the bank (A–H) using the cursor or hold [SETUP] + press pad [1]–[8] to switch banks
3
Select the pad number to configure
4
Set Category → Camera Control
5
Set Function → Choose from the table above
6
Set Value → Configure specifics (which camera, which preset, etc.)
7
Press [EXIT] → Repeat for each pad

7. 🎬 Scene Memories — 32 Total Snapshots

Manual p.76–80

What a scene saves: Video settings, audio settings, panel state, PinP positions, DSK settings, and more. Think of it as a "snapshot" of your entire board configuration.

Quick Save (Scenes 1–8)

1
Press [MODE] until it shows "MEMORY"
2
Long-press MEMORY button [1]–[8] to save

Blue = has saved data · Light blue = currently selected · Unlit = empty

Quick Recall (Scenes 1–8)

1
Press [MODE] → "MEMORY"
2
Short-press MEMORY button [1]–[8] to recall

Full Save/Recall (Scenes 1–32)

[MENU] → Scene Memory → Save/Load

  • Save: Select "Save" → choose destination number → confirm OK
  • Load: Select scene number → press [VALUE] to recall
  • Initialize: Select "Initialize" → choose scene → confirm
  • Name Edit: Rename scenes (up to 8 characters)

Scene Transition Effects

[MENU] → Scene Memory
SettingWhat It Does
Fade TimeHow long the transition takes when recalling a scene
Mix/Wipe = ONApply transition effect between scenes
PinP&Key = ONInset screen fades in smoothly
DSK = ONGraphics/captions fade in smoothly

Expand to 24 Buttons

[MENU] → Scene Memory → Button Assign → Number of MEMORY Button → 24

This maps PGM/A buttons [1]–[8] to scenes 9–16, and PST/B buttons [1]–[8] to scenes 17–24.

8. ⚡ Macros — 100 Automated Operations

Manual p.81–88

A macro records up to 10 operations and replays them with one button press. You can chain macros within macros for complex sequences. Perfect for repeatable church service transitions.

Creating a Macro

[MENU] → Macro → Edit

1
Select the macro number to edit
2
Select "Add Function" from the operation list
3
Choose the operation (from table above)
4
Set parameters for the operation
5
Set Timing: After Previous (sequential) or Same As Previous (simultaneous)
6
Repeat for up to 10 operations per macro

🔴 Live Recording a Macro (Fastest Method)

1
Open the macro editing screen
2
Select "Rec" → Press [VALUE]
3
Perform the operations live on the board — each operation is automatically captured
4
Select "Apply" to confirm

💡 Shortcut: Long-press a MACRO button to start recording directly!

Executing a Macro

  • Macros 1–8: Press [MODE] → "MACRO" → press MACRO button [1]–[8]
  • Macros 1–100: [MENU] → Macro → Execute → select number
  • Expand to 24 buttons: [MENU] → Macro → Button Assign → Number of MACRO Button → 24

Church Production Macro Examples

Macro 1: "Sermon Start"

  1. Camera Preset Recall → Pulpit close-up (Preset 2)
  2. DSK On → Lower third "Pastor [Name]"
  3. Audio Input Level → Boost pulpit mic
  4. Wait → 5 seconds
  5. DSK Off → Remove lower third

Macro 2: "Hymn Transition"

  1. Camera Preset Recall → Wide choir shot (Preset 4)
  2. PGM Take → Switch to choir camera
  3. Audio Input Level → Boost choir mics
  4. Audio Input Mute → Mute pulpit mic

Macro 3: "Stream Safety"

  1. Output Fade → Fade to black
  2. DSK Source → "We'll Be Right Back" still image
  3. DSK On → Show holding card
  4. Audio Output Mute → Mute all output

9. 📋 Sequencer — 1,000-Step Automation

Manual p.89–94

The sequencer chains scene memories, macros, and video switches into a linear timeline. Press [NEXT] to advance through each step — like a show cue list. Up to 1,000 steps.

What You Can Sequence

FunctionDescription
PGM TakeSwitch the final output video
Scene MemoryRecall any of the 32 scenes
MacroExecute any of the 100 macros

Running the Sequencer

1
Long-press SEQUENCER [ON] → Sequencer activates (button lights up)
2
Press [NEXT] → Executes first function. Button blinks during execution, stays lit when done.
3
Press [NEXT] again → Advances to next step
4
Press [PREVIOUS] → Go back one step
5
Long-press SEQUENCER [ON] again → Turn off

Auto Sequence (Hands-Free)

Each step can have a timing value:

ValueBehavior
PausePauses. Press [NEXT] to continue manually.
AutoImmediately executes the next step
1–120secWaits X seconds, then auto-advances
TIP Enable Repeat Execute ([MENU] → Sequencer → Repeat Execute → ON) to loop the sequence continuously — great for pre-service loops.

[MENU] → Mix/Wipe

  • Mix (Dissolve): Smooth crossfade between sources
  • Wipe: Various patterns (horizontal, vertical, diagonal, circle, etc.)
  • Cut: Instant switch (press [CUT] button)
  • Auto: Automated transition at set speed (press [AUTO] button)

11. 🎨 DSK — Downstream Keyer (Graphics Overlay)

Manual p.41–48

DSK overlays graphics (lower thirds, titles, logos, lyrics) on top of the final output. This is the key feature James mentioned — your primary graphics role.

Key Concepts

  • Source: What image/video gets overlaid (HDMI input, SDI input, still image, video player)
  • Key Type: Luminance (bright parts show) or Chroma (color-based, like green screen)
  • Still images with alpha: Load PNG images with transparency for clean overlays
  • Graphics Presenter: Companion software that sends graphics via HDMI for DSK compositing

12. 🖼️ Picture-in-Picture (PinP)

Manual p.38–40

  • 2 independent PinP layers (PinP&Key 1 and PinP&Key 2)
  • Each can be positioned, sized, and cropped independently
  • Supports key compositing (transparency) on the inset screen
  • Source: Any HDMI/SDI input, still image, or video player

14. 📡 Live Streaming & Recording

Manual p.66–75

Built-In Streaming Support

  • YouTube Live — direct integration
  • Facebook Live — direct integration
  • Twitch — direct integration
  • Custom RTMP — any streaming service
  • SRT — low-latency professional streaming (input AND output)

Recording
